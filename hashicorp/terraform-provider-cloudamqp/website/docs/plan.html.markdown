# subscription plans

Table below shows subscription plans available for CloudAMQP. `Lemur`is free of charge, for full price list see [cloudamqp](https://www.cloudamqp.com/plans.html). `Lemur`and `Tiger` are shared instances and share underlying hardware with other instances. They are also limited to which CloudAMQP provider resources that can be used. Further information on availability on each resource page.

Name | Plan | Type
---- | ---- | ----
Little lemur    | lemur   | shared
Tough Tiger     | tiger   | shared
Big Bunny       | bunny   | dedicated
Roaring Rabbit  | rabbit  | dedicated
Power Panda     | panda   | dedicated
Awesome Ape     | ape     | dedicated
Heavy Hippo     | hippo   | dedicated
Loud Lion       | lion    | dedicated
